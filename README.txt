QMPlot

QMPlot is a mathematical function plotter written in C++ using Qt toolkit.

Status

The project is no longer developed or maintained. I leave the source code here only for reference.
Original project was hosted at SourceForce since 2009 but I have moved it to GitHub: https://github.com/piotrdz/qmplot.
